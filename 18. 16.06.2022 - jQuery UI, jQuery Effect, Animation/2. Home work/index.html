Практическая часть

Разработайте решение задач с использованием библиотеки jQuery. Можете использовать библиотеку jQueryUI для построения интерфейса пользователя. Каждую задачу размещайте на отдельной странице, на главной странице разместите текст задания.

Задача 1. Требуется разработать классы для работы с расписанием отправлений поездов дальнего следования. Для каждого поезда указывается: номер поезда, станция назначения, время отправления.

Требуется разработать классы для работы с расписанием отправлений поездов дальнего следования. Для каждого поезда указывается: номер поезда, станция назначения, время отправления.

Хранить данные о поездах и вокзале в локальном хранилище, в формате JSON.

Функционал для разработки:

  • инициализация коллекции сведений о поездах (не менее 12 поездов);
  • добавление поезда в расписание
  • удаление поезда из расписания
  • изменение сведений о поезде в расписании
  • вывод сведений по всем поездам, упорядоченный по времени отправления
  • вывод сведений по всем поездам, упорядоченный по пункту назначения;
  • выделение сведений по поезду с запрошенным интервалом времени отправления;
  • выделение сведений по тем поездам, которые следуют до запрошенной станции назначения.

При упорядочивании сведений исходную коллекцию поездов не менять, работайте с копией этой коллекции.

Задача 2. Для трех вариантов русско-английский текста (выбирайте случайным образом при загрузке страницы и по клику на командную кнопку) найдите и выделите слова текста (средствами jQuery), начинающиеся и оканчивающиеся на гласные буквы. Поиск вести регистрозависимо или регистрнезависимо – задавать режим поиска радиокнопками. Запишите найденные слова и количество их вхождений в коллекцию объектов, выведите эту коллекцию на страницу.

Задача 3. Реализуйте анимацию средствами jQuery – перемещение произвольного небольшого изображения (изображение должно быть фоном блока div) в ограниченном прямоугольнике. Предусмотрите кнопки для запуска и останова движения по замкнутой траектории – движение по кругу (эллипсу). Также предусмотрите кнопки для ручного перемещения изображения (влево, вниз, вправо, вверх). Такие же перемещения должны быть назначены на клавиши A, S, D, W. Дополнительно: реализуйте перетаскивание фигуры в пределах блока.
