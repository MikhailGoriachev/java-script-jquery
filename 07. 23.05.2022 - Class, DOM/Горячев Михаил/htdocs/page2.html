Задача 2. Данные о погоде

Задача 2. Спроектировать класс в синтаксисе ES6+ для представления данных о погоде: температура, давление, влажность, скорость и направление ветра, графическое отображение атмосферных явлений (ясно, облачно, дождь, и т.д. – не более 5). Определите метод формирования строки для вывода данных в разметку. Переопределите метод toString() для простой вывода в консоль. Создайте массив данных о погоде за неделю, выведите его на страницу. По командам от кнопок выводите данные о погоде, упорядоченные (только при выводе, порядок элементов в исходном массиве не менять): по убыванию температуры, по возрастанию давления, по убыванию скорости ветра. По командам от кнопок выделяйте самые ветреные и самые тихие дни. Также должна быть кнопка для вывода исходного массива без выделений элементов.
